## Introduction to sing-box

[sing-box](https://github.com/SagerNet/sing-box) is an open source proxy software, it can change your ip address to access the internet with the help of a server, for example, if your country does not support [chatgpt](https://chatgpt.com/), you can use this software, sing-box software is similar to v2rayN, clash for window, clash.meta, [hiddify](https://hiddify.me/), [v2box](https://v2box.pro/), [shadowrocket](https://shadowrocket.ink), [Karing](https://karing.biz), [flclash](https://flclash.xyz),[mahsang](https://mahsang.pro), surge, [stash](https://apps.apple.com/us/app/stash-rule-based-proxy/id1596063349), etc., but it has many new features. For example, it is free to download from the Apple software store, no need to pay to download, sing-box supports many protocols, such as shadowsocks, vmess, [vless](https://velss.app), reality, Trojan, hysteria2 and other protocols.

## What platforms does sing-box support?

[sing-box](https://sing-box.info/) is a free cross-platform proxy software (The universal proxy platform), and the client supports use on Windows, Mac, Android, iOS, Linux, Apple TV and other platforms.
