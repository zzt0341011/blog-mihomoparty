# Руководство по использованию GUI.for.SingBox


[Графический интерфейс для SingBox](https://github.com/GUI-for-Cores/GUI.for.SingBox) — это мощное программное обеспечение сетевого прокси для Windows от стороннего производителя Sing-Box, которое просто в использовании и имеет графический интерфейс. Поддерживает большинство прокси-протоколов, представленных в настоящее время на рынке, таких как Shadowrockets, vmess, [vless](https://vless.app/ru), Reality, Trogan, Hysteria2 и другие протоколы.

## Первый шаг — скачать и установить

- [Нажмите здесь, чтобы загрузить версию GUI.for.SingBox для Windows.](https://sing-box.info/soft/GUI.for.SingBox-windows-amd64.zip)
- После скачивания разархивируйте файл и вы получите исполняемый файл GUI.for.SingBox.exe и папку data.


![Руководство по использованию версии GUI.for.SingBox для Windows1](https://sing-box.info/img/GUI.for.SingBox使用教程1.png)

## Шаг 2. Откройте исполняемый файл GUI.for.SingBox.exe.


![Руководство по использованию версии GUI.for.SingBox для Windows2](https://sing-box.info/img/GUI.for.SingBox使用教程2.png)

- Откройте его в первый раз, нажмите «Центр плагинов-плагинов», найдите преобразование узла и подмагазин, эти два плагина, установите их!
- После установки плагина нажмите «Обзор», нажмите «Быстрый старт» и введите узел аэропорта.


![Руководство по использованию версии GUI.for.SingBox для Windows3](https://sing-box.info/img/GUI.for.SingBox使用教程3.png)

![Руководство по использованию версии GUI.for.SingBox для Windows4](https://sing-box.info/img/GUI.for.SingBox使用教程4.png)

## Шаг 3. Введите соединение по подписке, затем нажмите, чтобы открыть системный прокси, и все готово.

- Например, введите ссылку ниже, и вы сможете временно использовать ее бесплатно.
```
https://bjly.mene.lol:5269/hui/D^Y^WiB0V.D^Y^WiB0V
```

![GUI.for.SingBox使用教程5](https://sing-box.info/img/GUI.for.SingBox使用教程5.png)

![GUI.for.SingBox使用教程6](https://sing-box.info/img/GUI.for.SingBox使用教程6.png)

- Нажмите на контроллер, а затем нажмите «Выбор узла». Вы можете вручную выбрать узел для подключения или позволить программному обеспечению выбрать его автоматически.

![GUI.for.SingBox使用教程7](https://sing-box.info/img/GUI.for.SingBox使用教程7.png)
