# GUI.for.SingBox使用教程

[GUI for SingBox](https://github.com/GUI-for-Cores/GUI.for.SingBox)是第三方**sing-box**强大的windows网络代理软件，简单易用、拥有图形界面。支持目前市面的大多数的代理协议，比如shadowsocks、vmess、[vless](https://vless.app/zh)、reality、trojan、hysteria2等协议。

## 第一步、下载安装

- [点击这里下载GUI.for.SingBox](https://sing-box.info/soft/GUI.for.SingBox-windows-amd64.zip)
- 下载完以后，解压文件，会得到一个可执行文件GUI.for.SingBox.exe，和一个文件夹data。


![GUI.for.SingBox使用教程1](https://sing-box.info/img/GUI.for.SingBox使用教程1.png)

## 第二步、 打开可执行文件**GUI.for.SingBox.exe**


![GUI.for.SingBox使用教程2](https://sing-box.info/img/GUI.for.SingBox使用教程2.png)

- 第一次打开，点击插件-插件中心，找到节点转换和sub-store，这两个插件，安装！
- 安装完插件以后点击概览，点击快速开始，输入机场节点
- 如果没有机场订阅节点，可以点击这里使用nekory-3.26，用免费节点

![GUI.for.SingBox使用教程3](https://sing-box.info/img/GUI.for.SingBox使用教程3.png)

![GUI.for.SingBox使用教程4](https://sing-box.info/img/GUI.for.SingBox使用教程4.png)

## 第三步 然后点击启动内核，再点击左上角的系统代理，就可以了

![GUI.for.SingBox使用教程5](https://sing-box.info/img/GUI.for.SingBox使用教程5.png)

![GUI.for.SingBox使用教程6](https://sing-box.info/img/GUI.for.SingBox使用教程6.png)

- 点击控制器，再点击节点选择，可以手动选择要连接的节点，也可以让软件自动选择

![GUI.for.SingBox使用教程7](https://sing-box.info/img/GUI.for.SingBox使用教程7.png)
